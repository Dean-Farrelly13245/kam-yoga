## Quick Start - Deploy to Netlify

### 1. Push to Git
```sh
git add .
git commit -m "Configure for Netlify deployment"
git push origin main
```

### 2. Connect to Netlify
1. Go to https://netlify.com
2. Click "Add new site" → "Import existing project"
3. Connect repository
4. Netlify auto-detects settings from `netlify.toml`

### 3. Add Environment Variables
In Netlify dashboard:
- Site settings → Environment variables
- Add `VITE_SUPABASE_URL`
- Add `VITE_SUPABASE_ANON_KEY`

### 4. Deploy
- Trigger deploy
- Note Netlify URL
- Update Supabase `APP_URL` secret
- Configure Stripe webhook

- [ ] Book a class
- [ ] Complete payment
- [ ] Admin login works

## Troubleshooting

### If Build Fails
1. Check `package.json` has `"build": "vite build"`
2. Check environment variables in Netlify
3. Check build logs in Netlify dashboard
4. Test locally: `npm run build`

### If Site Works but Assets 404
1. Check `vite.config.ts` base path
2. Check `App.tsx` basename
3. Clear Netlify cache and redeploy

### If Booking Fails
1. Check `APP_URL` in Supabase matches Netlify URL
2. Check Stripe webhook secret
3. Check Edge Function logs
